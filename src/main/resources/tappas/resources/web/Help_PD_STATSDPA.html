Differential PolyAdenylation results

The DPA results data table displays the statistical analysis results combined with expression and annotation data. The following table columns are displayed by default:

  • Gene - gene identifier as provided in the annotation file.
  • DPA Result - DPA for differential included, NOT DPA otherwise.
  • Q-Value - adjusted significance value obtained via the combination of p-values from DEXSeq.
  • Distal Condition - condition or time-point numbers (as named in the design file), where usage of the distal polyA site is favored.
  • Distal and Proximal Expression values per Condition - for each of the conditions or time-points, the distal and proximal polyA expression values are displayed. Expression in these cases results from the sum of expression values of all the isoforms contributing to the distal or proximal polyA sites, respectively. Check documentation at tappas.org for further information on polA site definition.
  • PolyA switching - polyA switching is called when the predominant polyA site used changes between conditions or time points, i.e. when the sum of expression of distal-expressing isoforms is higher than the sum of proximal-expressing isoforms in one condition but not in the other, or vice-versa.
  • ΔDistal PolyA Site Usage (ΔDPAU) - proportion of change in distal polyA site usage across conditions or time-points (absolute value).
  • Gene Description - gene description, if any, as provided in the annotation file.
  • Distal and Proximal mean expression - average expression for distal or proximal polyA sites across conditions or time points.
  • Chr - chromosome.
  • Strand - chromosome strand.
  • Isoforms - number of isoforms for this gene.
  • Coding - protein coding status: YES, if there are any coding transcripts, NO otherwise.
