Co-DFI Associations - data drill down

The Co-DFI associations drill down data table displays all the genes containing both selected DFI feature IDs, based on the original row where the drill down was requested. The following table columns are displayed by default:

  • Gene - gene identifier as provided in the annotation file.
  • Favored (Feature ID 1) - group/time-point favored for the first feature of the pair, based on expression levels.
  • Favored (Feature ID 2) - group/time-point favored for the second feature of the pair, based on expression levels.
  • Gene Description (add using + button) - gene description, if any, as provided in the annotation file
