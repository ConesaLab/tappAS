DEA Results for Genes

The DEA Results data table for genes displays the statistical analysis results combined with expression and annotation data. The following table columns are displayed by default:

  • Gene - gene identifier as provided in the annotation file.
  • Gene Description - gene description, if any, as provided in the annotation file.
  • DEA Result - DEA result: DE, differentially expressed, or NOT DE.
  • Regulated (two-group comparisons only) - regulation if DE: UP/DOWN, or blank if Not DE.
  • Log2FC - log2 fold change value, log2(FC).
  • Condition1 MeanExpLevel - the mean of the normalized gene expression levels (obtained after collapsing expression of transcripts from the same gene) for all samples in condition 1.
  • Condition2 MeanExpLevel - the mean of the normalized gene expression levels (obtained after collapsing expression of transcripts from the same gene) for all samples in condition 2.
Note that "condition" is shown only in the case of two-group comparisons, while group name and time-point indicator are shown for single and multiple time-course designs.
  • Cluster (only for single and multiple time series) - cluster number that the gene was assigned to if DE.
NOISeq columns:
  • Probability - probability value for the given DE result (values between 0 and 1).
  • (1 - Probability) - 1 minus the probability value for given result. Provides the value used for the significance test.
edgeR columns:
  • p-Value - significance value obtained for DE testing.
  • FDR - False Discovery Rate. Provides the multiple-testing corrected significance value.
  • LogCPM - average log2 of Counts Per Million value for all groups/time-points.
  • Chr - chromosome where the gene is found.
  • Strand - chromosome strand.
  • Isoforms - number of isoforms for this gene.
  • Coding - protein coding status: YES, if there are any coding transcripts, or NO otherwise.
  • Annotation Feature Columns - use the 'Add annotation feature column...' menu selection.
