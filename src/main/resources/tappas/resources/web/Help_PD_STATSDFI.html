Differential Feature Inclusion results

The DFI Results data table displays the statistical analysis results combined with expression and annotation data. The following table columns are displayed by default:

  • Gene - gene identifier as provided in the annotation file.
  • Feature - feature category name as specified in the annotation file.
  • Feature ID - feature ID as specified in the annotation file.
  • Position - genomic position (start and end) for the analyzed feature.
  • DFI Result - DFI for differential included, NOT DFI otherwise.
  • Q-Value - adjusted significance value obtained via the combination of p-values from DEXSeq.
  • Major isoform switching - YES or NO, used to indicate if the most expressed variant (feature-including or feature-excluding) changed between conditions.
  • Favored Conditions/Time points - condition or time-point number (as named in the design file), where Feature ID is favored by expression levels.
  • ΔFeature Inclusion (ΔFI) - magnitude of the redistribution of feature inclusion levels across conditions or time-points.
  • Gene Description - gene description, if any, as provided in the annotation file.
  • Feature Description - feature ID description, if any, as provided in the annotation file.
  • Source - annotation source for the feature ID as specified in the annotation file.
  • Chr - chromosome.
  • Strand - chromosome strand.
  • Isoforms - number of isoforms for this gene.
  • Coding - protein coding status: YES, if there are any coding transcripts, NO otherwise.
  • Annotation Feature Columns - use the 'Add annotation feature column...' menu selection
