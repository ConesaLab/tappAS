Project Data Visualization: Expression Matrix Summary

The Expression Matrix Summary subtab contains multiple tables and visual display controls summarizing the transcripts expression matrix data. The display controls are described below by position, left to right.

Conditions data table - displays the expression matrix conditions and corresponding number of samples. The table displays the following columns for each condition and the total:

  • Condition - condition name as defined in the original expression matrix file
  • Samples - number of samples for given condition

Filtering pie chart and data table - the pie chart displays the proportions for filtered and not filtered transcripts. Filtered transcripts are further divided into:

  • No Annotation - transcripts filtered for having no annotations in the annotation file
  • LowCount/COV - transcripts filtered due to expression levels being too low or coefficient of variation limit exceeded
  • Transcripts List - transcripts filtered for not being in inclusion filter list or for being in exclusion filter list

The table displays the following columns for filtered, not filtered, and total transcripts:

  • Filtering - filtering type or not filtered
  • Count - number of transcripts of given type
  • % - percentage of total transcripts
