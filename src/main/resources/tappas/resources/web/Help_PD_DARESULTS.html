Combined Results for all Differential Analyses

The Combined Results data table displays all the Differential Analysis (DE and DIU for genes, transcripts and CDS, when previously performed) statistical analysis results combined with expression and annotation data. The following table columns are displayed by default:

Genes Section
  • Gene - gene identifier as provided in the annotation file
  • Gene Description - gene description, if any, as provided in the annotation file
  • DIU Result for Transcripts - DIU result: DIU, differentially spliced, NOT DIU, or blank for single isoform genes
  • DIU Result for Proteins - DIU result: DIU, differentially spliced, NOT DIU, or blank for non-coding or single protein genes
  • DEA Result - DEA result: DE, differentially expressed, or NOT DE
Proteins Section
  • DE - differentially expressed: comma separated list of all DE proteins
  • Total - total number of gene CDS found
  • Switching - YES for genes with CDS isoform switching, NO otherwise
Transcripts Section
  • DE - differentially expressed: comma separated list of all DE transcripts
  • Total - total number of gene transcripts
  • Switching - YES for genes with transcript isoform switching, NO otherwise
  • Chr - chromosome where the gene is present
  • Strand - chromosome strand
  • Isoforms - number of isoforms for this gene
  • Coding - protein coding status: YES, if there are any coding transcripts, or NO if not
  • Annotation Feature Columns - use the 'Add annotation feature column...' menu selection
