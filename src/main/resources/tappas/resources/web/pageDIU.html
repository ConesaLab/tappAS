Differential Isoform Usage can be assessed for transcripts or CDS (see image below). Just like when using transcripts, CDS-level DIU allows checking for differences in expression across conditions/time-points considering protein isoforms. CDS levels for DIU are calculated using the sum of expression of transcripts encoding the same coding region (i.e. protein). TappAS uses DEXSeq R package for DIU analysis.





Differential Isoform Usage Analysis Parameters


When using the application, all DIU parameters are described in the Help page which can be accessed via the Help button located at the bottom of the dialog window. Please be aware that you do not need to rerun the analysis to change the significance level value: a button to change the significance threshold and reassign DIU/Not DIU status is provided on the left hand-side menu. see Subtabs Menu Bar section for details.

DIU Results

The DIU results table is contained in the project data tab, located in the top panel (see image below). It includes basic information fields, such as gene and gene description, although more columns can be included through the (+) button on the right-hand side of the table. It also includes the results (DIU/NotDIU), Q-Value (i.e. multiple-testing adjusted indicator of significance), Total Usage Change, and Isoform switching. Major isoform switching is an indicator of whether the most expressed transcript or CDS, depending on the data type selected, changed between conditions or time-points. Total Usage Change measures the redistribution of expression across isoforms between conditions or time-points. The mean expression levels for each condition are shown for each gene.

When running the application, a description of all fields in the result table can be viewed using the Help button on the left-hand side menu.

DIU Results Summary

The DIU results summary can be accessed through the visualization button in the left hand-side menu (beside the results table), and will appear in the bottom tab panel (see image below). It includes both a summary pie chart of DIU/Not DIU and Switching/No Switching status, as well as a scatter plot representing gene log2(Fold Change) vs Total Usage Change.
