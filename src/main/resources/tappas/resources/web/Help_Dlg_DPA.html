Differential PolyAdenylation (DPA) Analysis parameters

To evaluate Differential PolyAdenylation (DPA), you must provide the following information:

  1. PolyAdenylation range
    Minimum value for differentiation - number of base pairs to call a different polyA site between isoforms. All alternative polyA sites in a gene's isoforms that are less than x base pairs apart (where x is the provided threshold) are collapsed and considered to be a result of stochastic polyA site variation.
  2. Analysis Method section
    Method - select which analysis method to use for DPA:
    • DEXSeq - The package is focused on finding differential exon usage using RNA-seq exon counts between samples with different experimental designs. It provides functions that allows the user to make the necessary statistical tests based on a model that uses the negative binomial distribution to estimate the variance between biological replicates and generalized linear models for testing.
    • maSigPro - Differential isoform expression usage for RNA-seq data in single and multiple time-course experiments. After summing the expression of the distal-expressing and proximal-expressing isoforms for the two defined polyA sites in each gene, tappAS uses the Iso-maSigPro functionality in the maSigPro R package. maSigPro applies linear regression to model gene expression and selects significant genes via a two-step algorithm. maSigPro returns lists of genes with statistically significant changes along time and across the different series.

  3. Analysis Parameters section
    Significance Level - specify the significance level required to call a gene as DPA (default: 0.05)
    Polynomial Degree - (time-course only) degree for the maSigPro regression model. The maximum allowed degree is time_points-1.
    Filtering - specify whether or not to filter minor isoforms prior to testing for Differential PolyAdenylation. Recommended to prevent spurious detection of DPA genes due to small usage changes in minor isoforms (i.e. very lowly expressed). Two methods are available:
    • Filtering by proportion - does not consider isoforms that represent a proportion of gene expression that is below the specified threshold (default: 0.2)
    • Filtering by fold expression difference - compares all isoforms to the major (i.e. the one with highest mean expression across conditions) and filters those with a fold difference above the specified threshold (default: 2)

    Max K Clusters - (time-course only) to show the trajectories in plots, maSigPro visualization applies clustering methods to group genes with similar distal polyA site usage trends, and summarizes this in the graphical display. This parameter determines the maximum number of clusters to classify genes into. Enable the "Use mclust for optimal number" option to use a method to dynamically determine the number of clusters.
