Project Data Visualization: Transcript Exons Distribution per Structural Category

The Transcript Exons Distribution per Structural Category chart displays the distribution of the number of transcript exons, in the form of a box plot, per structural category. The structural categories are shown on the x-axis and will vary based on the annotation file used. The number of transcript exons, for the box plots, is shown on the y-axis.

Don't forget to take advantage of mouseover and context-sensitive menus for additional information and functionality.
