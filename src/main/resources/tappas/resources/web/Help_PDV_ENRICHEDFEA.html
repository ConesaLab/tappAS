Project Data Visualization: FEA Enriched Features

The FEA Enriched Features chart displays a graphical summary of functional features enrichment.

  • The initial chart displays the top 5 more significantly enriched feature IDs for each feature category. This display can be changed using the left hand-side Options menu, either changing the number of top features within the category, manually selecting the enriched feature IDs to be shown, or providing an enriched features list. On the latter, users may select non-significant features for the purpose of comparison.
  • The chart plot shows Feature IDs on the y-axis, and the number of genes in the test list containing a given feature ID on the x-axis.
  • The feature ID dot properties convey additional information, reflected on the chart legend. Namely, dot color reflects the feature category to which the feature ID belongs to, and the size is determined by -log10(adjusted p-value), i.e. smaller p-values result in bigger the circles, indicating higher significance.

    • Don't forget to take advantage of mouseover and context-sensitive menus for additional information and functionality.
