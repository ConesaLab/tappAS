Row Selection Editor parameters

To filter the current results table, users need to provide the following information:

  1. Action type:
    Add to existing selections - elements matching the selected criteria and comparison will be added to the current results table.
    Remove from existing selections - elements matching the selected criteria and comparison will be removed from the current results table.

  2. Criteria field: this field will show a list of all available columns in the table, including, but not limited to, significance (p-value and adjusted p-value), IDs (feature ID, gene ID, transcript ID) and quantitative variables (log2FC, Total Change, Expression Level, etc.). The list will change depending on the results table to be filtered.

  3. Comparison: specifies the logical operation to be performed on the selected criteria field. Including, but not limited to:
    Numeric comparisons - greater than, equal to, less than or equal to, etc.
    Within/outside numeric range comparisons.
    Text and pattern comparisons - target elements containing or not containig a certain text pattern, a word, etc. Exact match comparisons can be performed by loading values from a .tsv file containing IDs, names, keywords, etc.

    4. NOTE: using these filtering options, several nested filtering steps can be performed. Once you have performed your first filtering, open this window again to add/remove elements from the extant selection.
