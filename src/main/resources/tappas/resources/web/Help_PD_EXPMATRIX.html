Project Data: Expression Matrix

The Expression Matrix data table displays the original transcript-level expression data along with transcript specific filtering information. The following table columns are displayed by default:

  • Id - transcript Id as provided in the expression matrix
  • Filtered - transcript filtered status:
    'NO' if not filtered,
    'YES - LowCount/COV' if filtered as a result of low counts or exceeding the coefficient of variation cutoff,
    'YES - Transcripts List' if filtered as a result of the transcripts list,
    'YES - No Annotation' if filtered as a result of having no annotation in the annotation file
Condition 1 Samples Section
  • 1 - expression level for first sample
  • ...
  • n - expression level for last sample
Condition 2 Samples Section
  • 1 - expression level for first sample
  • ...
  • n - expression level for last sample
  • Annotation Feature Columns - use the 'Add annotation feature column...' menu selection
