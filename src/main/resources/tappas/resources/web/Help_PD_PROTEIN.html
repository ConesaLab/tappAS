Project Data: Proteins

The Proteins data table combines the transcript-level expression data with the functional annotation for the project. The following table columns are displayed by default:

  • Protein - protein identifier as provided in the annotation file
  • Transcript(s) - transcript(s) encoding this protein, could be more than one, as provided in the annotation file
  • Gene - transcript's gene identifier as provided in the annotation file
  • Gene Description - gene description, if any, as provided in the annotation file
  • Condition1 MeanExpLevel - the mean of the normalized protein expression levels for all samples in condition 1
  • Condition2 MeanExpLevel - the mean of the normalized protein expression levels for all samples in condition 2
The expression level for a protein is the sum of the normalized levels for all transcripts encoding the protein.
  • Length - protein sequence length in amino acids
  • Annotation Feature Columns - use the 'Add annotation feature column...' menu selection
