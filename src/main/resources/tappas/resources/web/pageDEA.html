Differential Expression Analysis (DEA) performs statistical testing to determine whether a given difference in read counts between conditions/time-points, is significant or just due to random variations. You may run DEA at the gene, transcript and CDS levels (see image below). CDS and gene expression levels are calculated using the sum of transcript expression (in the case of CDS, aggregating transcripts that code for the same predicted protein). Users may select NOISeq or edgeR as the underlying R package for the analysis.





Differential Expression Analysis Parameters


When using the application, all DEA parameters are described in the Help page which can be accessed via the Help button located on the bottom left of the dialog window.

NOISeq

NOISeq provides "differential expression between two experimental conditions with no parametric assumptions".

You may view the documentation and installation instructions at:
https://www.bioconductor.org/packages/release/bioc/html/NOISeq.html

edgeR

EdgeR provides "differential expression analysis of RNA-seq expression profiles with biological replication. Implements a range of statistical methodology based on the negative binomial distributions, including empirical Bayes estimation, exact tests, generalized linear models and quasi-likelihood tests".

You may view the documentation and installation instructions at:
https://www.bioconductor.org/packages/release/bioc/html/edgeR.html


DEA Results

The DEA results are contained in the project data tab, displayed in the top panel (see image below). It includes basic information fields about gene, transcripts, or CDS (depending on the DEA data type selected) as well as the DEA test results, DE/NotDE and Up/Down regulation if DE, Probability or P-Value, (depending on the R package used), and the Log2 of the fold change (Log2FC). However, users may add more columns by clicking on the (+) button on the right hand side of the table. The mean expression levels for each condition per gene, transcript or CDS are also shown.

Be aware that you do not need to rerun the analysis to change the significance level value: a button is provided on the left hand side menu bar to change the significance level value and recalculate the DE/NotDE results. When running the application, a description of all fields in the results table can be viewed using the subtab Help button on the left-hand side menu bar, and a DE Results Summary can be opened by clicking on the visualization button in this same menu.
