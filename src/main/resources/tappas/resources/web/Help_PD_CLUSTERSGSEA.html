GSEA Multidimensional Enrichment Summary table

The summary table contains the following columns:

  • Node (feature ID) - GO term IDs as shown in the annotation file.
  • Name - GO term description as shown in the annotation file.
  • Feature - biological aspect described by the GO term, i.e. C = Component, P = Process or F = Function..
  • Over-Pvalue (one column for each of the two input ranked lists) - over representation p-value obtained by the MDGSA method, for each of the input ranked lists.

To generate the multidimensional GSEA clusters graph, use the visualization button on the left hand-side menu beside the table.

  • Non-Test Items - number of genes NOT in Test group that are annotated for that GO term.
Row selection () is not supported for these tables but sequential row id (#) columns are displayed
