Export Data

The following options are provided to export data:

Data - select data to export. List of selections will vary based on the data table. The most commonly provided options are:
  • Table Rows - export all the data for each row in the table
  • Items list (IDs only) - export only the ID for each row item - where item can be a transcript, gene, etc.
  • Items ranked list (IDs and P-Values) - export only the ID and P-Value for each row item - where item can be a transcript, gene, etc.
Some data tables may provide other options.
Include all data - select to include all data rows in the table
Include only selected rows - select to include only the selected table rows. Selected rows are those having the row selection box checked.

The exported data will be saved using a tab separated value (TSV) file format and will contain a header line starting with a hash character, e.g. "#ID   P-Value".
