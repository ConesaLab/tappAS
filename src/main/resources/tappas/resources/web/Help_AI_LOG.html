App Info: Log

The application log displays application messages. The full date is displayed when the application is started or when the date changes. Each message entry displays the time followed by the application message, see sample below:

2018-5-30 Fri 11:20
11:20:36.978 - tappAS 0.99.07 using Java 8.0.141-b14

In addition to the normal messages, the following message types may be displayed:

  • [DBG] - debug message, requires debug message option to be enabled, see below
  • [WARN] - warning message, the application encountered a problem that you should be aware of
  • [ERROR] - error message, the application encountered a problem which caused some action to fail
Options to select all the text and copy it are provided via context-sensitive menu by right-clicking with the mouse anywhere on the page. Note that the log file display is cleared when the application is restarted. However, the actual log file retains the previous information. If the log file gets too large, some of the older information will be removed to bring the file size down.
