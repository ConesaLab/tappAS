Gene Data Visualization: Annotation File Data

The Annotation file data table displays all the functional annotation entries in the annotation file for the given gene. The table columns correspond to the fields defined in the Generic Feature Format Version 3 (GFF3):

  • Transcript - transcript id, referred to as "seqid" in GFF3
  • Source - database name where feature information was extracted from
  • Feature - annotation feature type, referred to as "type" in GFF3
  • Start - start coordinate, 1 based relative to feature location
  • End - end coordinate
  • Score - not used
  • Strand - strand, + or -
  • Phase - not used
  • Attributes - list of feature attributes in the format of key=value separated by semicolons
