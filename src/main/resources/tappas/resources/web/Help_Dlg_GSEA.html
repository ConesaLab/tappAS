Gene Set Enrichment Analysis parameters

To run Gene Set Enrichment Analysis (GSEA), you must provide the following information:

  1. Analysis Information Section
    Name - specify unique name for analysis

  2. Data Type Section
    Data Type - select which data type to use for GSEA:
    • Genes - use gene expression data (obtained after collapsing expression of isoforms from the same gene)
    • Transcripts - use transcript expression data (obtained after collapsing expression of isoforms with the same CDS)
    • CDS - use CDS expression data

  3. Analysis Method Section
    GOglm - applies GSEA analysis with prior correction of length bias, i.e. the tendency to detect differential expression more frequently in longer genes, which causes categories containing longer genes to be more frequently detected as enriched.
    MDGSA - Multidimensional Gene Set Enrichment Analysis. Can accomodate one or more ranking indices.
    • Umivariate - performs GSEA based on a single ranked statistic.
    • Multidimensional - performs GSEA based on two ranked statistics, for instance, p-value lists for DE and DIU genes. This allows for finding gene set enrichments due to the combined effect of two transcriptomic phenomena. In this case, the Annotation Feature menu is inactive, since the analysis is only available for Gene Ontologies.

  4. Items Ranked List Section
    Ranked List - select from one of the options provided (will vary based on data type).
    For genes:
    • Differential Isoform Usage (Transcripts) - DIU genes based on transcripts
    • Differential Isoform Usage (Proteins) - DIU genes based on proteins
    • DIU Results (Transcripts - including single isoform genes) - DIU results based on transcripts
    • DIU Results (Proteins - including single isoform genes) - DIU results based on proteins
    • Differential Expression Analysis Results - differentially expressed genes
    • Use list from file - specify a list to use in file selection control below
    For transcripts:
    • Differential Expression Analysis Results - differentially expressed transcripts
    • Use list from file - specify a list to use in file selection control below
    For CDS:
    • Differential Expression Analysis Results - differentially expressed CDS
    • Use list from file - specify a list to use in file selection control below
    You must have previously generated the data for some of the options, e.g. DIU and DEA results. In addition, note that, for multidimensional MDGSA, two ranked lists need to be provided.

  5. Annotation Feature(s) Section
    Use annotation file feature(s) - select the annotation feature(s) to include in the GSEA. Note that the tree contains one branch per annotation source and each branch can be expanded to show all the features it contains. You may select all features for a given annotation source or expand and select specific features only. You may ONLY include features from a SINGLE annotation source.
    Use custom annotation sets, matching data type, in GMT file format (*.gmt) - specify file path in file selection control provided below.

  6. Analysis Parameters Section
    Significance Level - specify the significance level required to consider significant enrichment (default: 0.05)
