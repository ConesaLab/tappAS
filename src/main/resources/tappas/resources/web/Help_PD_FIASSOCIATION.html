Co-Differential Feature Inclusion Results (feature associations)

The co-DFI Results data table displays summary information for the association of paired feature IDs with genes where both feature IDs are DFI. The following table columns are displayed by default:

  • Feature ID 1 - feature ID 1 name as specified in the annotation file.
  • Feature ID 2 - feature ID 2 name as specified in the annotation file.
  • Genes with both DFI Feature IDs - number of genes where both Feature IDs are positive for differential inclusion.
  • Co-inclusion - number of genes for which inclusion of both feature IDs is favored in the same condition.
  • Mutual exclusion - number of genes for which inclusion of both feature IDs is favored in opposite condition.
  • Source 1 - annotation source for Feature ID 1 as specified in the annotation file.
  • Feature 1 - feature category name for Feature ID 1 as specified in the annotation file.
  • Source 2 - annotation source for Feature ID 2 as specified in the annotation file.
  • Feature 2 - feature category name for Feature ID 2 as specified in the annotation file.
