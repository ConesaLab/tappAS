DFI Summary - data drill down

The DFI summary drill down data table displays all the genes containing the selected DFI feature ID, based on the original row where the drill down was requested. The following table columns are displayed by default:

  • Gene - gene identifier as provided in the annotation file
  • Position - feature ID genomic position, if genomic position method was used to define DFI, blank otherwise
  • Favored - group/time-point favored based on expression levels
  • Gene Description (add using the + button) - gene description, if any, as provided in the annotation file
