Project Properties

The project properties are displayed in a data table format as property:value pairs. The following properties are shown:

  • Name - project name as entered
  • Internal Id - internal application project ID
  • Species - species specified for the input data
  • Annotation Source - source for annotation file: user or application provided
  • Annotation Type - if application annotation file, the reference type is displayed: RefSeq or Ensembl
  • Annotation File - file path for original annotation file used
  • Expression File - file path for original expression matrix file used
  • Condition 1 Name - name for condition 1 as specified in the expression matrix file header
  • Condition 1 Samples - number of samples for condition 1 as provided in the expression matrix file
  • Condition 2 Name - name for condition 2 as specified in the expression matrix file header
  • Condition 2 Samples - number of samples for condition 2 as provided in the expression matrix file
  • Expression Level Filtering - filtering option status if specified for the input data: Yes or No
  • Low Count Cutoff - low count CPM cutoff value if specified for the input data
  • Coefficient of Variation Cutoff - coefficient of variation cutoff percentage as specified for the input data
  • Additional Transcript Filter - transcripts list filter if specified for the input data
  • Transcript Filter List File - transcripts list file path if specified for the input data
This is not a standard data table so no table menu, search, or filtering functionality is provided.
