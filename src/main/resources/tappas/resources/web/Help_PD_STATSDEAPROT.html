DEA Results for CDS

The DEA Results data table for proteins displays the statistical analysis results combined with expression and annotation data. The following table columns are displayed by default:

  • Protein - protein identifier as provided in the expression matrix.
  • Transcript(s) - transcript(s) encoding this protein, could be more than one, as provided in the annotation file.
  • Gene - transcript's gene identifier as provided in the annotation file.
  • Gene Description - gene description, if any, as provided in the annotation file.
  • DEA Result - DEA result: DE, differentially expressed, or NOT DE.
  • Regulated (two-group comparisons only) - regulation if DE: UP/DOWN, or blank if Not DE.
  • Log2FC - log2 fold change value, log2(FC).
  • Condition1 MeanExpLevel - the mean of the normalized CDS expression levels (obtained after collapsing expression of transcripts with the same CDS) for all samples in condition 1.
  • Condition2 MeanExpLevel - the mean of the normalized CDS expression levels (obtained after collapsing expression of transcripts with the same CDS) for all samples in condition 2.
Note that "condition" is shown only in the case of two-group comparisons, while group name and time-point indicator are shown for single and multiple time-course designs.
  • Cluster (only for single and multiple time series) - cluster number that the CDS was assigned to if DE.
NOISeq columns:
  • Probability - probability value for the given DE result (values between 0 and 1).
  • (1 - Probability) - 1 minus the probability value for given result. Provides the value used for the significance test.
edgeR columns:
  • p-Value - significance value obtained for DE testing.
  • FDR - False Discovery Rate. Provides the multiple-testing corrected significance value.
  • LogCPM - average log2 of Counts Per Million value for all groups/time-points.
  • Length - protein sequence length in amino acids.
  • Proteins - no. of proteins encoded by the gene that the protein comes from.
  • Annotation Feature Columns - use the 'Add annotation feature column...' menu selection
