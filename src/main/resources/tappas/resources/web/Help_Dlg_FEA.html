Functional Enrichment Analysis parameters

To run Functional Enrichment Analysis (FEA), you must provide the following information:

  1. Analysis Information Section
    Name - specify unique name for analysis

  2. Data Type Section
    Data Type - select which data type to use for FEA:
    • Genes - use gene expression data (obtained after collapsing expression of isoforms from the same gene)
    • Transcripts - use transcript expression data
    • CDS - use CDS expression data (obtained after collapsing expression of isoforms with the same CDS)

  3. Item Lists Section
    Test List - List of items (gene/transcript/CDS) whose enrichment will be evaluated. Select from one of the options provided (will vary based on data type).
    For genes:
    • Differential Isoform Usage (Transcripts) - DIU genes based on transcripts
    • Not Differential Isoform Usage (Transcripts) - not DIU genes based on transcripts
    • Differential Isoform Usage (Proteins) - DIU genes based on proteins
    • Not Differential Isoform Usaged (Proteins) - not DIU genes based on proteins
    • Differentially Expressed - differentially expressed genes
    • Not Differentially Expressed - not differentially expressed genes
    • Use list from file - specify a list to use in file selection control below
    For transcripts:
    • Differentially Expressed - differentially expressed transcripts
    • Not Differentially Expressed - not differentially expressed transcripts
    • Use list from file - specify a list to use in file selection control below
    For CDS:
    • Differentially Expressed - differentially expressed CDS
    • Not Differentially Expressed - not differentially expressed CDS
    • Use list from file - specify a list to use in file selection control below
    Note that you must have previously generated the data for some of the options, e.g. DIU and DEA results.
    Background List - List of items (gene/transcript/CDS) against which enrichment will be tested. Select from one of the options provided (will vary based on data type).
    For genes:
    • All Genes - all genes in project
    • All multiple isoforms genes - all genes in projects with multiple isoforms
    • Use list from file - specify a list to use in file selection control below
    For transcripts:
    • All Transcripts - all transcripts in project
    • Use list from file - specify a list to use in file selection control below
    For CDS:
    • All CDS - all CDS in project
    • Use list from file - specify a list to use in file selection control below

  4. Annotation Feature(s) Section
    Annotation Feature(s) - select the annotation feature(s) to include for FEA. Note that the tree contains one branch per annotation source and each branch can be expanded to show all the features it contains. You may select all features for a given annotation source or expand and select specific features only. You may include features from a several annotation sources.

  5. Analysis Parameters Section
    Statistical Test - select which statistical test to use:
    • Wallenius - approximates the true distribution of numbers of members of a category amongst DE genes by the Wallenius non-central hypergeometric distribution.
    • Sampling - uses random sampling to approximate the true distribution and uses it to calculate the p- values for over (and under) representation of categories. Must specify the number of random samples to be calculated if so.
    • Hypergeometric - Hypergeometric assumes there is no bias in power to detect differential expression at all and calculates the p-values using a standard hypergeometric distribution. Useful if you wish to test the effect of selection bias on your results. CAUTION: Hypergeometric should NEVER be used for producing results for biological interpretation.
    Use w/o Category - select whether to use items not containing specific features or not:
    • Yes - include items even if they do not contain a specific feature being tested. These genes will count towards the total number of genes outside the feature being tested.
    • No - only include items containing specific features tested. These genes will be ignored in the calculation of p-values (default).
    Note that only items that have at least one of the features selected will be included to begin with. However, it is possible for an item to have some of the selected features but not all.
    Significance Level - specify the significance level required for Differential Expression (default: 0.05)
