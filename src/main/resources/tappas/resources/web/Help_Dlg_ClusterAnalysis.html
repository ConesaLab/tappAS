Cluster Analysis Parameters

The Cluster Analysis is a Java implementation of the DAVID Bioinformatics fuzzy heuristic gene clustering algorithm. To run Cluster Analysis, you must provide the following information:

Similarity Term Overlap - minimum number of overlapped annotation feature IDs between two items in order to be qualified for kappa calculation (default: 4)
Kappa Threshold - minimum kappa value to be considered biologically significant (default: 3.5). Higher values will result in higher quality of functional classification, that is, fewer nodes with fewer members.
Initial Seed Node Members - minimum initial number of items in a seeding node (default: 4). Lower values will generate a greater number of small size nodes.
Final Seed Node Members - minimum number of items in a final node (default: 4). This value works in conjunction with the previous parameter to control the minimum node size.
Node Linkage Threshold - controls how nodes are merged (default: 0.5). Two nodes sharing a higher percentage of items than the specified threshold, will be merged into a single node. High values typically give sharper separation of nodes with items within the node having a tighter association.

"Items" refer to genes, transcripts, or proteins depending on the selected data type for the analysis.
