UTR Lengthening Results

The UTRL results data table displays the results after weighting UTR lengths of each gene's isoforms (considering the provided bp threshold for differentiation of UTR lengths) by the proportion of expression of the isoform (i.e. isoform A expression / total gene expression), resulting in the wUTR value. The following table columns are displayed by default:

  • Gene - gene identifier as provided in the annotation file.
  • Expression-weighted UTR3 or UTR5 - a column group is created for 3' and 5'UTRs, each containing the wUTR value of the specified UTR (3 or 5) in each condition or time-point.
  • Gene Description - gene description, if any, as provided in the annotation file.
  • Distal and Proximal mean expression - average expression for distal or proximal polyA sites across conditions or time points.
  • Chr - chromosome.
  • Strand - chromosome strand.
  • Isoforms - number of isoforms for this gene.
  • Coding - protein coding status: YES, if there are any coding transcripts, NO otherwise.
