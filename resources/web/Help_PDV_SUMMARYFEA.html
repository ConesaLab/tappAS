Project Data Visualization: FEA Summary

The FEA Summary subtab contains multiple plots, along with supporting tables, summarizing the FEA results. FEA can be run using genes, CDS, or transcript data (hence the word Item is used refer to features, depending on the data type used). Note that enrichment can be tested for multiple functional categories simultaneously, all ow which will appear in both the tables and the summary plots. The display controls are described below by position, left to right.

Left panel: Items Summary pie chart and data table - the pie chart displays the proportions of test and non-test items.

  • The number of test items is determined by the number of features positive for that annotation in the test list.
  • The number of non-test items is calculated by subtracting the number of test features from the total number of features in the background list.

    • Below the pie chart, the following table columns are displayed for test, non-test, and total items:

    • Item - item type
    • Count - number of items of given type
    • % - percent of total items of given type

    Middle panel: Enriched Feature Count the table displays the total number of significantly enriched feature IDs in the test list per functional feature category.

    The following table columns are displayed:

    • Feature - feature category name.
    • Enriched Feature IDs - number of significantly enriched feature IDs for a given feature category.
    • Total Feature IDs - total number of feature IDs for given feature

    Right panel: Horizontal Bar Chart the horizontal bar chart displays the percentage of enriched feature IDs (x axis) for each feature on the y axis. The total number of feature IDs tested for each functional feature category is used to calculate the percentages (i.e. the data in the table in the middle panel).
