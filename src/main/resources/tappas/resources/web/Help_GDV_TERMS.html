Gene Data Visualization: Annotation Features Diversity

The Annotation Features diversity table displays a cross-reference of functional annotation features and isoforms for the given gene. The rows display the annotation features and the columns the gene isoforms. The cell values contain the number of features for the intersecting row and column. The following table columns are displayed:

  • Feature ID - annotation feature ID
  • Name/Description - feature ID name/description
  • [Transcript/Protein] - transcript name, and protein if available. Multiple columns, one for each transcript

The cross-reference display provides a quick and simple way to visually observe the features presence and variability among the gene isoforms. You may also select to only show varying feature, see below.

  - menu with the following selections:
  • Annotation Features Filter... - filter annotation features included in the table
  • Show ONLY varying features - check to display only the features that vary across isoforms
  - export data or images to file. See Export Data and Images section in Overview
