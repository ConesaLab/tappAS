Differential Expression Analysis Parameters

To run Differential Expression Analysis (DEA), you must provide the following information:

  1. Data Type Section
    Data Type - select which data type to use for DEA:
    • Genes - use gene expression data (obtained after collapsing transcript isoform expression for each gene)
    • Transcripts - use transcript expression data
    • CDS - use CDS expression data (obtained after collapsing transcript isoform expression by coding region similarity)

  2. Analysis Method Section
    Method - select which analysis method to use for DEA:
    • edgeR - Differential expression analysis of RNA-seq expression profiles with biological replication. Implements a range of statistical methodology based on the negative binomial distributions, including empirical Bayes estimation, exact tests, generalized linear models and quasi-likelihood tests.
    • NOISeq - Differential expression between two experimental conditions with no parametric assumptions.

    • maSigPro - Differential expression for RNA-seq data in single and multiple time-course experiments. Applies linear regression to model gene expression and selects significant genes via a two-step algorithm. First, responsive genes are identified by fitting a generic regression model with time as quantitative variable and series as dummy variables. Second, step-wise regression is applied on selected genes to adjust models and identify gene-specific variation patterns. maSigPro returns lists of genes with statistically significant changes along time and across the different series.

  3. Analysis Parameters Section (two-group comparisons)
    Replicates - select the type of replicates provided in the expression matrix, if any (NOISeq only):
    • Biological - select for biologically distinct samples
    • Technical - select for same samples measured using different techniques
    Significance Level - specify the significance level required for Differential Expression (default: 0.05)
    Fold Change - specify the Fold Change required for Differential Expression (default: 2, NOISeq only)

  4. Analysis Parameters Section (single and multiple time course designs)
    Polynomial Degree - degree for the regression model. The maximum allowed degree is time_points-1.
    Significance Level - specify the significance level required for Differential Expression (default: 0.05)
    R^2 cutoff - required level of the goodness of fit of the regression model. This parameter is between 0 and 1. Higher values indicate well fitted models. MaSigPro documentation recommends values between [0.4,0.8].
    Max K Clusters - To show the trajectories in plots maSigPro visualization applies clustering methods to group genes with similar trends and summarize the graphical display. This parameter determines the maximum number of clusters to classify genes into. Enable the "Use mclust for optimal number" option to use a method to dynamically determine the number of clusters.
