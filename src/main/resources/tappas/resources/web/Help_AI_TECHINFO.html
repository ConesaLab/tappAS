App Info: Technical Information

The application technical information subtab displays relevant technical information about applications, user, and computer. Version numbers, number of processors and type, etc. are displayed along with a section showing the environment seen by all scripts run by the application. This information can be useful when troubleshooting script problems and is likely to be different from what you would see from the terminal.

Options to select all the text and copy it are provided via context-sensitive menu by right-clicking with the mouse anywhere on the page.
