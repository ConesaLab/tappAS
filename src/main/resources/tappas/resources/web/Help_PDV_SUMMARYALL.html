Project Data Visualization: General Summary

The General Summary subtab contains a table and multiple visual display controls summarizing the expression data and functional annotation for the project. The display controls are arranged in two rows and are described below by row, top to bottom, and position, left to right.

Data Summary table - displays the total counts of all expressed items. The following table columns are displayed:

  • Transcripts - total number of expressed transcripts after filtering
  • Proteins - total number of expressed proteins after filtering
  • Genes - total number of expressed genes after filtering

Transcripts and Proteins per Gene bar chart - displays the distribution, per gene, of transcripts and proteins. Note that it is possible for a gene to have no proteins but it must have some transcripts.

Transcript Structural Categories pie chart - displays the proportion of each structural category based on the number of transcripts assigned in the annotation file.

Annotation Sources bar chart - displays the number of entries contained in the annotation file for each annotation source.

Expression Levels PCA plot - displays a PCA plot of the expression levels for all samples in both conditions.

Be aware that there are two rows of visualization controls in this subtab - in case you don't notice the scroll bar.
