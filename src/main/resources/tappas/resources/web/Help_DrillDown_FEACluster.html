FEA Cluster Drill Down Data

The FEA cluster drill down data table displays all the genes contained in the selected cluster based on the original row where the drill down was requested. The following table columns are displayed by default:

  • Gene - gene identifier as provided in the annotation file
  • In Test List - indicates if the gene was included in the test list: YES or NO
  • Node(s) - list of all the nodes in the cluster that given gene is a member of
  • Gene Description - gene description, if any, as provided in the annotation file
