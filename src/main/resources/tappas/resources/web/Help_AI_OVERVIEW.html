App Info: Overview

The application overview provides a comprehensive description of the application. The overview is divided into sections which can be accessed directly via the contents menu, on the right side of the page, or sequentially using the left and right arrow content navigation buttons located on top of the page, to the left of the contents menu. If the application window is not wide enough, the contents menu will not be shown and you must use the content navigation buttons. You also have the option to hide the contents menu to maximize the display area. The following sections are provided in the overview:

  • Introduction - High level application overview including application requirements sub section
  • Projects - Use of projects in the application including sub sections on input data, normalization, annotation file format, and expression matrix file format
  • Application Interface - User interaction with the application including sub sections on application layout, menus, tabs and subtabs, subtab menu bar, data tables, and data visualization controls
  • Application Tips - Tips for getting the most out of the application
  • Data Visualization - Application data visualization
  • Data Drill Down - Data drill down functionality via context-sensitive menus
  • Export Data and Images - Exporting data from tables and images from visualization controls
  • Ad Hoc Query - Simple data search and filtering using row selection queries
  • Feature Diversity Analysis (FDA) - Qualitative analysis of functional diversity in the annotated transcriptome at the gene level
  • Differential Expression Analysis (DEA) - DEA for genes, transcripts and CDS
  • Differential Isoform Usage Analysis (DIU) - DIU for transcripts or CDS
  • Differential Feature Inclusion (DFI) - Differential inclusion analysis for annotation features
  • Differential PolyAdenylation (DPA) - Differential polyA site usage analysis
  • 3' and 5' UTR Lengthening (UL) - Expression-weighted analysis of UTR length across groups/time points
  • Enrichment Analyses - Functional Enrichment Analysis and Gene Set Enrichment Analysis (uni and multidimensional)

The overview pages are meant to be accessed via the content menu and/or content navigation control buttons. A browser like back page button, in the form of a 'Go Back' menu selection, can be accessed via context-sensitive menu by right-clicking with the mouse anywhere on the page.
