Project Data Visualization: Annotation Source Details

The Annotation Source Details subtab displays additional information for any given annotation source. There are two tables displayed: the Features table on the left and the Feature IDs table on the right. When you highlight a feature on the Features table, by clicking on the corresponding row, the Feature IDs for that feature are shown in the Feature IDs table. Note that you may hold the Shift-key down and select multiple features.

Features data table - displays features summary information for given annotation source. The following table columns are displayed:

  • Feature - annotation feature as specified in the annotation file
  • Feature IDs - number of unique annotation feature IDs for the given source:feature
  • Entries - total number of entries in the annotation file for the given source:feature
  • Transcripts - total number of transcript containing the given source:feature

Feature IDs data table - displays feature IDs summary information for selected annotation feature(s) in the Features table. The following table columns are displayed:

  • Feature - annotation feature as specified in the annotation file
  • Feature ID - feature ID as specified in the annotation file
  • Entries - total number of entries in the annotation file for the given source:feature:featureID
  • Transcripts - total number of transcript containing the given source:feature:featureID

Don't forget to take advantage of mouseover and context-sensitive menus for additional information and functionality.
