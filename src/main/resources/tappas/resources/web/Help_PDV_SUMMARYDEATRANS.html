Project Data Visualization: Transcripts DEA Summary

The Transcripts DEA Summary subtab contains multiple visual display controls, along with a supporting table, summarizing the transcripts DEA results. The display controls are described below by position, left to right.

Transcripts DEA Results pie chart and data table - the pie chart displays the proportions for differentially and not differentially expressed transcripts. The table displays the following table columns for coding, non-coding, and total transcripts:

  • Transcript - transcript type
  • DE - number of differentially expressed transcripts of given type
  • Not DE - number of NOT differentially expressed transcripts of given type
  • Total - total number of transcripts of given type

Transcripts Log2 Fold Change Distribution bar chart - displays the L2FC distribution for all transcripts. The bins for the fold changes are shown on the x-axis with the down-regulated transcripts on the left part and the up-regulated transcripts on the right. Square brackets, [], are used to denote values that are inclusive and parentheses, (), for values that are exclusive. For example, a bin with a "[5, 10)" label contains values from 5 to less than 10, i.e. inclusive of 5 but exclusive of 10. Note that up and down regulation is based on the condition comparison you selected for the analysis. The total number of transcripts for a given bin is determined by the height of the bars along the y-axis. Negative values are used for down regulated counts, on the y-axis, for visualization purposes only; counts are always positive.
