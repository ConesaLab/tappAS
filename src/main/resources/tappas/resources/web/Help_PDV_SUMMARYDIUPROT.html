Project Data Visualization: DIU genes (CDS level) summary

The Gene CDS DIU Summary subtab contains multiple visual display controls, along with two summary plots of CDS-level DIU results. The display controls are described below by position, left to right.

Gene CDS DIU Results pie chart - the pie chart displays the proportions for genes with differential isoform usage and genes that are not DIU. Proportions of genes with and without major isoform switching are also represented.

Gene Summary DIU Results plot - log2FC of gene expression is represented (x axis) vs Total Change (y axis). To more easily detect genes with high differential isoform usage levels, switching and DIU status are indicated by dot color and filling.

Note that only genes that code for multiple proteins are included in this analysis.
