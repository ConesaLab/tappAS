GSEA Enriched Features Display Filter

The GSEA enriched features filter allows you to select which features to display so that you can see the ones that you are interested in. To select which enriched features to display, two display options are provided:

Top 'n' enriched feature IDs for each feature - select to display the top 'n' feature IDs for each feature.
n - specify the value for 'n' (1 to 50)
All manually selected enriched features - select to manually specify, in the tree control, which feature IDs to display.
Select Enriched Feature(s) to display - select the enriched features to include in the display. Note that the tree contains one branch per feature and each branch can be expanded to show all the feature IDs it contains. You may select all feature IDs for a given feature or expand and select specific feature IDs only. Feature IDs are sorted by descending adjusted p-value, with the lowest value on top.
Clear All - click to clear all selected features
Check All - click to check, select, all features
Select enriched features by list - provide a list of feature IDs (as a single-column .tsv file) to show in the chart. You may select non-significant features for the purpose of comparison.

Note: The total number of feature IDs to display on the chart is limited to 100 for both display options.
