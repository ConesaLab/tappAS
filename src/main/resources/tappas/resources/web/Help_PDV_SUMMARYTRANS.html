Project Data Visualization: Transcripts Summary

The Transcripts Summary subtab contains multiple visual display controls, along with a supporting table, summarizing the transcript expression data and associated functional annotation. The display controls are described below by position, left to right.

Transcripts pie chart and data table - the pie chart displays the proportions for coding and non-coding transcripts. The table displays the following table columns for coding, non-coding, and total transcripts:

  • Transcript - transcript type
  • Count - number of transcripts of given type
  • % - percentage of total transcripts

Transcript Lengths box plot - displays the length distribution, in nucleotides, for all transcripts.

Transcript Expression Levels density plot - displays the expression levels distribution, using ggplot2 and geom_density, for each condition.

Transcript Structural Categories pie chart - displays the proportion of each structural category based on the number of transcripts assigned in the annotation file.
