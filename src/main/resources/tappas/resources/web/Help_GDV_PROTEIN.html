Gene Data Visualization: Proteins View

The Proteins View subtab provides data visualization for all gene proteins, if any, including annotation features. Several options are provided in the options menu, see below, to control the visualization display.




Proteins View Subtab

A. Gene Information Header
A single line containing the gene name and description.
B. Additional Information
A single line specifying the view type and alignment mode.
C. Relative Position Ruler Marks
Position ruler marks - relative, zero based, amino acid position marks only shown if 'Show Ruler' option is checked, see below.
D. Protein Visualization Tracks
One track per gene protein. Protein name is displayed on left side followed by visual display of protein along with annotation features, see Legend for details. The protein size, in amino acids, is displayed on the right side of the track.
E. Legend
Legend box for all annotation feature symbols displayed.

Be aware that, except for DOMAINS, all annotation features are shown as symbols with each symbol positioned on the center of the full span of the feature. Even though this is adequate for most features given their small size, some features span a larger area which is not reflected on the display. Don't forget to take advantage of mouseover and context-sensitive menus for additional information and functionality.

 - menu with the following selections:
  • Annotation Features Filter... - filter annotation features included in the display
  • Show ONLY varying features - check to display only the features that vary across proteins
  • Show PROVEAN Score - check to display PROVEAN score, if available
  • Show Aligned - check to show proteins aligned properly based on genomic position
  • Show Ruler - check to display genomic position ruler marks
  • Display Size - size of display, similar to font sizes for text: Small, Medium (default), or Large
  • Sort by - order in which proteins are displayed from top to bottom:
    ID - alphabetically by transcript id
    Length- based on transcript length in nucleotide
    Expression Level - based on transcript's mean expression levels
    PROVEAN Score - based on PROVEAN score, if available
  • Descending Order - check to display values in descending order. Applies to all sorting options
