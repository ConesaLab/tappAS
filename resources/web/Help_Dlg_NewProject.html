New Project Parameters

To create a new project, you must provide the following information:

  1. Project Information
    Name - unique project name

  2. Annotation Data
    Species - select from species list supported by the application or choose "Other" if not available.
    Use application's annotation file - select to use the default annotation file provided in the application for the selected species. This option is only available for supported species.
    Specify annotation's file location - select to provide your own annotation file. Enter the file's path and name manually or use the file selection button to the right.

  3. Experiment Design and Data
    Experiment Type - select experiment type:
    • Two-Group Comparison - basic case-control experiment with replicates
    • Single Series Time-Course - single experimental group with samples taken at multiple time points
    • Multiple Series Time-Course - multiple experimental groups with samples taken at multiple time points
    Design File - experimental design file path and name.
    Matrix Data File - expression matrix file path and name.
    Expression Values Filter - select filtering to apply to the expression matrix, if any:
    • Low count values cutoff - Cutoff value, in counts per million (CPM), in a sample. Set to eliminate transcripts with low counts.
    • Coefficient of variation cutoff - Cutoff percentage value for the coefficient of variation per condition. Set to eliminate transcripts with inconsistent expression values.
    Transcripts Filter - select optional transcripts filtering:
    • Do not filter transcripts - no additional filtering
    • Inclusion list - only include transcripts contained in inclusion list specified below
    • Exclusion list - exclude transcripts contained in exclusion list specified below
    Transcripts List - inclusion/exclusion, based on previous selection, transcripts list file path and name
