Project Data Visualization: maSigPro Cluster Visualization for time-course designs

The time-course analyses performed by maSigPro include a clustering option where genes with similar patterns of expression over time (gene, transcript or CDS, but also polyA site usage or weighted UTR length) are grouped. See package documentation and analysis help dialogues for additional information.

% Cluster table - shows the list of genes belonging to the selected cluster. Use the button below the list to visualize a different cluster.

% Cluster median profile - shows the median expression of all genes in the cluster. Total gene no. is specified in the plot title. If multiple time series are available, expression of the clustered genes in each of the series is plotted in a different color. Dotted lines indicate the overall change trend inferred from cluster-level expression.

% View All Cluster Plots - opens a supplementary window where all cluster profiles are shown simultaneously.
