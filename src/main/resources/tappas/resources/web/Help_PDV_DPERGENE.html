Project Data Visualization: Distribution per Gene

The Distribution per Gene chart displays the distribution of transcripts and proteins per gene. The bins for the number of transcripts and proteins per gene are shown on the x-axis. The percentage of the total genes for a given bin is determined by the height of the bars along the y-axis. Note that it is possible for a gene to have no proteins but it must have some transcripts.
