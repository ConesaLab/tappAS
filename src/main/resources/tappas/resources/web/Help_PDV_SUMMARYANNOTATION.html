Project Data Visualization: Annotation Summary

The Annotation Summary subtab contains an annotation summary table and pie chart summarizing functional annotation entries in the annotation file. The display controls are described below by position, left to right.

Annotation Summary data table - displays the number of entries found in the annotation file for each annotation feature. The following table columns are displayed:

  • Source - annotation source as specified in the annotation file
  • Feature - annotation feature as specified in the annotation file
  • Feature IDs - number of unique annotation feature IDs for the given source:feature
  • Entries - total number of entries in the annotation file for the given source:feature

Annotation Sources pie chart - displays the proportion of each annotation source based on the number of entries in the annotation file.

You may see additional information for any given annotation source via context-menu selection provided in both the table and pie chart.
