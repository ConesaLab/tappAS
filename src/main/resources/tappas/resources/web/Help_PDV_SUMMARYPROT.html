Project Data Visualization: Proteins Summary

The Proteins Summary subtab contains multiple visual display controls, along with a supporting table, summarizing the proteins expression data and associated functional annotation. The display controls are described below by position, left to right.

Proteins pie chart and data table - the pie chart displays the proportions for single and multiple transcript encoded proteins. The table displays the following table columns for single transcript, multiple transcript, and total proteins:

  • Protein - protein type
  • Count - number of proteins of given type
  • % - percentage of total proteins

Protein Lengths box plot - displays the length distribution, in amino acids, for all proteins.

Protein Expression Levels density plot - displays the expression levels distribution, using ggplot2 and geom_density, for each condition.

The expression level for a protein is the sum of the normalized levels for all transcripts encoding the protein.
