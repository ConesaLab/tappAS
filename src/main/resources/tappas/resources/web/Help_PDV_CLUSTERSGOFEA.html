FEA Clusters Data Visualization: Gene Ontology (GO)

The Gene Ontology subtab displays a directed acyclic graph (DAG) of GO terms. The graph displays the relationship among terms in the different domains. The terms displayed in the graph are based on the GO terms contained in the selected FEA cluster being displayed.

Legend:

  orange border is used to indicate GO slim terms
  yellow rectangle with red border is used to indicate root domain term
  green rectangle used to indicate GO term contained in cluster
  gray line is for links showing 'is_a' relationship
  blue line is for links showing 'part_of' relationship

A tool bar located below the chart provides controls to interact with the display:

Show - select which domain to show the GO terms graph for:
  • BP - biological process
  • MF - molecular function
  • BP - cellular component
for - select which cluster to use for building the graph:
  • [Specific Cluster] - select a specific cluster. List of clusters will vary based on clustering results
display - select how 'deep' to show graph:
  • Low level graph - shows only a few levels down from the root for simplicity
  • Full graph - shows full graph so that all annotated GO terms for the selected cluster are included
  • Up to level 'n' - shows graph up to level 'n', where 'n' is 1 through 4
Note that you may get additional levels, from the one selected, in some cases due to the relationship among terms.

Don't forget to take advantage of mouseover and context-sensitive menus for additional information and functionality.

  - menu with the following selections:
  • Show GO term id - check to display the GO term id in the graph nodes
  • Show GO term name - check to display the GO term name in the graph nodes
  • Show GO term id and name - check to display the GO term id and name in the graph nodes
