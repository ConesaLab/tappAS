Gene Data Visualization: Expression Charts

The Expression Charts subtab contains multiple line charts showing expression levels per condition for the given gene and associated transcripts, and proteins if any. All charts display the two conditions on the x-axis and the corresponding expression levels on the y-axis. Use the mouseover feature, or refer to the legend, to identify which transcript or protein a given line belongs to. Note that the mouseover feature for line charts only works on the endpoints of the line. The charts are described below by position, left to right.

Gene Expression line chart - displays the gene expression level for both conditions.

Transcripts Expression line chart - displays the gene transcripts expression level for both conditions.

Proteins Expression line chart - displays the gene proteins, if any, expression level for both conditions.

The expression level for a gene is the sum of the normalized levels for all its transcripts
The expression level for a gene protein is the sum of the normalized levels for all gene transcripts encoding the protein.
