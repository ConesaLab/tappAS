Gene Data Visualization: Genomic View

The Genomic View subtab provides a genome browser-like view of the gene transcripts. Unlike the transcripts and proteins subtabs, a full view, including all introns and exons, is displayed. In addition, just like in a genome browser, the transcripts and genomic annotation features are all displayed in their own individual tracks. Several options are provided in the options menu, see below, to control the visualization display.




Genomic View Subtab

A. Gene Information Header
A single line containing the gene name and description.
B. Additional Information
A single line specifying the view type and chromosome name.
C. Chromosome Layout
A rectangular layout of the chromosome including G-bands, if available. The location of the gene on the chromosome is indicated by vertical red bar.
D. Genomic Position Ruler Marks
Genomic position ruler marks - genomic position marks only shown if 'Show Ruler' option is checked, see below.
E. Transcripts Visualization Track
One track per gene transcript. Transcript name is displayed on left side followed by visual display of introns and exons. The transcript size, in nucleotides, is displayed on the right side of the track. If the 'Show Additional Information' options is selected, see below, additional transcript information is displayed, if available, on the transcript name line.
F. Annotation Feature Track
One track per annotation feature. Feature name displayed on left side followed by visual display of features positioned accordingly along the transcripts. The number of feature tracks will vary based on the number of features included in the genomic section of the annotation file used.

Don't forget to take advantage of mouseover and context-sensitive menus for additional information and functionality.

  - menu with the following selections:
  • Show Details - check to display additional structural information, if available, for transcripts
  • Show Ruler - check to display genomic position ruler marks
  • Display Size - size of display, similar to font sizes for text: Small, Medium (default), or Large
  • Sort by - order in which transcripts are displayed from top to bottom:
    ID - alphabetically by transcript id
    Length- based on transcript length in nucleotide
    Expression Level - based on transcript's mean expression levels
    PROVEAN Score - based on PROVEAN score, only applicable for coding transcripts and availability
  • Descending Order - check to display values in descending order. Applies to all sorting options
