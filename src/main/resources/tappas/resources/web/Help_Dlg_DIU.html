Differential Isoform Usage analysis parameters

To run Differential Isoform Usage (DIU) analysis, you must provide the following information:

  1. Data Type Section
    Data Type - select which gene data type to use for DIU:
    • CDS - use gene CDS expression data (obtained after collapsing transcript isoform expression by coding region similarity)
    • Transcripts - use gene transcript expression data

  2. Analysis Method Section
    Method - select which analysis method to use for DIU:
    • DEXSeq - The package is focused on finding differential exon usage using RNA-seq exon counts between samples with different experimental designs. It provides functions that allows the user to make the necessary statistical tests based on a model that uses the negative binomial distribution to estimate the variance between biological replicates and generalized linear models for testing.

    • maSigPro - Differential isoform expression usage for RNA-seq data in single and multiple time-course experiments. Uses the Iso-maSigPro functionality in the maSigPro R package. maSigPro applies linear regression to model gene expression and selects significant genes via a two-step algorithm. First, responsive genes are identified by fitting a generic regression model with time as quantitative variable and series as dummy variables. Second, step-wise regression is applied on selected genes to adjust models and identify gene-specific variation patterns. maSigPro returns lists of genes with statistically significant changes along time and across the different series.

  3. Analysis Parameters Section
    Significance Level - specify the significance level required for Differential Isoform Usage (default: 0.05)
    Polynomial Degree - (only time-course designs) degree for the maSigPro regression model. The maximum allowed degree is time_points-1.
    Filtering - specify whether or not to filter minor isoforms prior to testing for Differential Isoform Usage. Recommended to prevent spurious detection of DIU genes due to small usage changes in minor isoforms. Two methods are available:
    • Filtering by proportion - does not consider isoforms that represent a proportion of gene expression that is below the specified threshold (default: 0.2)
    • Filtering by fold expression difference - compares all isoforms to the major (i.e. the one with highest mean expression across conditions) and filters those with a fold difference above the specified threshold (default: 2)
