Project Data Visualization: UTR Lengthening (UTRL) Summary

The UTRL Summary subtab contains tables and plots summarizing the analysis results.

% UTRL statistical test result table - (for two-group comparisons) shows the Wilcoxon test p-value for 3' and 5' UTRs, indicating whether a significant change in UTR lengths exists between conditions.

% wUTR difference violin plot - (for two-group comparisons) shows the distributions of the increment in weighted UTR between the two conditions (can be positive or negative depending on the direction of change) for each tested gene. Both 3' and 5'UTR results are represented.

The following requirements apply to this analysis:

  • For a gene to be included in this analysis, it must have multiple isoforms with different UTR lengths, separated by a length equal to the bp threshold provided for analysis.
