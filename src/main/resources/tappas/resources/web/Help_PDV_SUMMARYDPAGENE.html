Project Data Visualization: Differential PolyAdenylation (DPA) Summary

The DPA Summary subtab contains multiple plots and charts summarizing the analysis results. These may vary depending on the significance threshold chosen. If you wish to recalculate the summary, close the subtab, change the significance level using the left hand-side bar menu, and open it again.

% Genes DPA Results pie chart showing the proportion of DPA genes vs the proportion of not DPA genes.

% Distal DPA Results pie chart showing the percentage of genes that are predominantly distal-expressing on each of the conditions (i.e. the Distal Condition column), as well as the percentage that undergo polyA site switching in each case.

DPA Summary Table - shows the number of coding and non-coding genes that are detected as DPA, and the total numbers in each category.

Genes Log2 Fold Change Distribution bar chart - histogram of Log2FC values (binned) for distal and proximal site expression for all analyzed genes.

DPA heatmap - for each significantly DPA gene, the polyA site usage proportion in each analyzed condition is shown. The heatmap is divided between polyA switching and not switching genes. Allows an intuitive visualization of the polyA site usage differences, as well as the most distal-expressing condition.

The following requirements apply to this analysis:

  • For a gene to be included in this analysis, it must have multiple isoforms and contain at least two different polyA sites. For genes with more than two sites, a polyA site definition algorithm is applied (see web documentation).
