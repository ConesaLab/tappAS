Project Data Visualization: GSEA Clusters Graph

The Multidimensional GSEA Clusters Graph shows a network in which GO terms are clustered according to their functional similarity, i.e. the similarities in the biological processes (P), components (C) or broad molecular functions (F) that they describe. The network is generated for the GO terms in the Multidimensional Enrichment Summary. In this network:

  • Nodes correspond to clustered GO terms. A label beside the node shows the GO term description.
  • Edges indicate the existence of semantic similarity between two GO terms.
  • Node color - in this network, each node will be colored in the form of a pie chart in order to compare the different levels of significance reported by MDGSA method for each GO term in each ranked list. Pie chart area is defined by the relative -log10(over p-value) to measure the differential effect of each of the variables measured in the ranked lists in the genes that are annotated under that GO term (i.e. a proxy to biological function). For instance, if DIU and DE genes are selected as input ranked lists, different orange vs blue color proportions would point towards preferential regulation by Differential Isoform Usage or by Differential Expression mechanisms. However, different interpretations may be produced given that this analysis can be run for any kind of user-defined ranked list.
  • Dynamic information - use the mouse to scroll the graph to view full GO term names, GO IDs, number of genes significantly enriched at the top of the ranked lists, and p-values in each of the MDGSA result columns.
  • Node size indicates the number of genes annotated for that GO term that were detected as significantly highly ranked by MDGSA (i.e. by combining the two test lists).

NOTE: Users may drag nodes and cluster labels to arrange the clusters and export high quality images of the clustering results (see left hand-side panel "Export" button).
