The FDA analysis explores the diversity of annotation features in the transcriptome, and their potential to be differently regulated via AltTP mechanisms. Isoforms vary in structural and functional features among isoforms of the same gene. FDA identifies and measures the nature of the variability in a qualitative manner. For every annotation category, all pairwise comparisons between transcript isoforms from the same gene are performed to evaluate whether at least one isoform pair has variability in a feature, either in its annotated genomic position(s) (Positional Varying) or in the presence/absence of the annotated feature (Presence Varying). Note that, given its qualitative nature, FDA evaluates the annotated diversity only, and that expression values are not considered in this analysis. The Positional Varying approach compares features by genomic position, i.e. by mapping features to genomic coordinates and classifying them as varying if coordinates are not equivalent between gene isoforms. Position disagreement is annotated when >9bp, that is, 3 amino acids, allowing for variability in prediction. In contrast, Presence Varying includes only presence/absence of annotation. For instance, NMD transcript status is based on differences in the transcript-level NMD label. In contrast, transcript attributes such as UTR length, CDS and polyA site positions, are examples of features where only positional evaluation is meaningful. However, a third group of features (such as Pfam domains or transmembrane regions) can be affected by AltTP via both complete skipping and partial disruption of the feature. In these and similar cases, both strategies can be used, and provide complementary insight on AltTP in the potential regulation of the functional or regulatory feature. Functional Diversity can be assessed by gene or by feature ID.



Gene-Level FDA

The Gene-Level Diversity analysis evaluates genes as a function of the structural, functional and regulatory features that are modulated by AltTP and reports the diversity rate for each category. As described above, depending on the category and its relationship to the functional properties of a transcript or protein, Functional Diversity is evaluated using a Positional Varying strategy or a Presence Varying strategy.





Gene-Level FDA Analysis Parameters


When using the application, all FDA parameters are described in the Help page which can be accessed via the Help button located on the bottom left of the dialog window.

Gene-Level Results

After running the analysis, the FDA results table will appear in the project data tab, located in the top panel (see image below). Each table row displays the diversity results for a given gene, for all its annotated categories. The result columns are grouped into transcript and protein categories, depending on whether the feature refers to transcript or protein functions. Each functional category column displays the VARYING or NOT VARYING label for the corresponding gene, indicating whether at least one of its isoforms lacks a functional feature from this category. Blank row cells indicate no features from that particular category were present in any of the gene's isoforms.

ID-Level FDA

The ID-Level Feature Diversity Analysis analysis identifies specific functional and regulatory elements (i.e. by feature ID instead of source/functional category) varying across isoforms from the same gene. The diversity status of each ID can also be evaluated via Positional, Presence Varying or both. Feature-Level Diversity Analysis evaluates global variation rates for the feature IDs contained in a given category and tests their enrichment in varying status using a Fisher's Exact Test. Significance values are then corrected using the Benjamini-Hochberg method for multiple testing correction.





ID-Level FDA Analysis Parameters


When using the application, all FDA parameters are described in the Help page, which can be accessed via the Help button located at the bottom of the dialog window.




ID-Level FDA Results

The ID-Level FDA Results Table is contained in the project data tab located in the top tab panel (see image below). Each table row displays the ID of a functional feature from the analyzed category (Description and Source columns can be added by clicking the (+) button on the right hand-side of the top panel). Columns indicate the number of genes where each feature was detected as VARYING or NOT VARYING, as well as the p-value and adjusted p-value (FDR) obtained from the significance test.

ID-Level FDA Combined Results

When an ID-Level FDA has been run twice for the same category using the genomic position and feature presence approaches, a combined results option will be unlocked in the Diversity menu (main menu bar). This is available for all categories that can be analyzed considering total skipping of the feature and partial disruption (for instance, PFAM Domains). The frequencies of both types of variation can be compared by generating this Combined Results plot.
