UTR Lengthening (UTRL) Analysis parameters

To evaluate changes in UTR Lengthening (UTRL), you must provide the following information:

  1. Lengthening range
    Minimum value for differentiation - number of base pairs to call a different UTR length between isoforms. All isoforms with UTR lengths differing less than x base pairs (where x is the provided threshold) are considered to have the same UTR length. Once UTR lengths for analysis have been selected, each length is weighted by the relative expression of the isoform (isoform expression / total gene expression), and sumed to provide a weighted 3' or 5' UTR length (wUTR).
  2. Analysis Method section
    Method - select which analysis method to use to test UTRL:
    • Wilcoxon - Available for two-group comparisons. Performs a Wilcoxon signed-rank test on the wUTR values between conditions, and provides a p-value indicating whether the change in UTR length is globally significant.

    • maSigPro - Available for time-course designs. Performs a gene-level analysis of wUTR values across time-points, and provides a p-value for each gene indicating whether the change in UTR length (weighted by each isoform's proportion of expression) is globally significant across time, for each of the series provided (in the case of multiple designs).

  3. Analysis Parameters section
    Significance Level - specify the significance level required to call a gene as significantly changing UTR length (for time-course desings) or to consider the global UTR length change between conditions significant (for two-group comparisons)
    Polynomial Degree - (time-course only) degree for the maSigPro regression model. The maximum allowed degree is time_points-1.
    Filtering - specify whether or not to filter minor isoforms prior to testing for UTR Lengthening. Recommended to prevent considering UTR lengths from minor isoforms (i.e. very lowly expressed). Two methods are available:
    • Filtering by proportion - does not consider isoforms that represent a proportion of gene expression that is below the specified threshold (default: 0.2)
    • Filtering by fold expression difference - compares all isoforms to the major (i.e. the one with highest mean expression across conditions) and filters those with a fold difference above the specified threshold (default: 2)

    Max K Clusters - (time-course only) to show the trajectories in plots, maSigPro visualization applies clustering methods to group genes with similar weighted UTR expression trends, and summarizes this in the graphical display. This parameter determines the maximum number of clusters to classify genes into, using wUTR values. Enable the "Use mclust for optimal number" option to use a method to dynamically determine the number of clusters.
