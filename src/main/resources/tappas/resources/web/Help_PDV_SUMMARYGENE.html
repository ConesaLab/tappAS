Project Data Visualization: Genes Summary

The Genes Summary subtab contains multiple visual display controls, along with supporting tables, summarizing the expression data and associated functional annotation. The display controls are described below by position, left to right.

Protein Coding pie chart and data table - the pie chart displays the proportions for coding and non-coding genes. Coding genes are further divided into single and multiple protein coding. The table displays the following table columns for non-coding, single and multiple proteins coding, and total genes:

  • Gene - gene type
  • Count - number of genes of given type
  • % - percentage of total genes

Transcripts pie chart and data table - the pie chart displays the proportions for single and multiple transcript genes. The table displays the following table columns for single transcript, multiple transcripts, and total genes:

  • Gene - gene type
  • Count - number of genes of given type
  • % - percentage of total genes

Gene Expression Levels density plot - displays the expression levels distribution, using ggplot2 and geom_density, for each condition.

The expression level for a gene is the sum of the normalized levels for all its transcripts
