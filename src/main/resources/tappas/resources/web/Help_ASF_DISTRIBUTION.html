Project Data Visualization: Annotation Source Features Distribution

The Annotation Source Features Distribution chart displays the feature(s) distribution for any given annotation source. The bins on the x-axis represent the number of annotation entries per transcript. The number of transcripts for a given bin is determined by the height of the bars along the y-axis.

A tool bar located below the chart provides control(s) to interactively change the display:

Show distribution for - select which feature to show the distribution data for:
  • <All Features> - all features combined
  • [Specific Feature] - select a specific feature. List of features will vary based on given annotation source.

Don't forget to take advantage of mouseover and context-sensitive menus for additional information and functionality.
