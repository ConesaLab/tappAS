Project Data: Transcripts

The Transcripts data table combines the transcript-level expression data with the functional annotation for the project. The following table columns are displayed by default:

  • Transcript - transcript identifier as provided in the expression matrix
  • Protein - transcript's protein identifier, if any, as provided in the annotation file
  • Gene - transcript's gene identifier as provided in the annotation file
  • Gene Description - gene description, if any, as provided in the annotation file
  • Condition1 MeanExpLevel - the mean of the normalized transcript expression levels for all samples in condition 1
  • Condition2 MeanExpLevel - the mean of the normalized transcript expression levels for all samples in condition 2
  • Chr - chromosome
  • Strand - chromosome strand
  • Coding - protein coding status: YES or NO
  • Length - transcript sequence length in nucleotides
  • Structural Category - structural category classification as provided in the annotation file
  • Structural Attributes - structural attributes as provided in the annotation file
  • Annotation Feature Columns - use the 'Add annotation feature column...' menu selection
