DIU Results for CDS

The DIU Results data table for CDS displays the statistical analysis results combined with expression and annotation data. The following table columns are displayed by default:

  • Gene - gene identifier as provided in the annotation file.
  • Gene Description - gene description, if any, as provided in the annotation file.
  • DIU Result - DIU for genes that are positive for differential isoform usage, NOT DIU otherwise.
  • Q-Value - adjusted significance value obtained via the combination of exon-level p-values from DEXSeq.
  • Major isoform switching - YES or NO, used to indicate if the most expressed isoform changed between conditions.
  • Total Change - magnitude of the redistribution of isoform expression across groups or time-points.
  • Condition1 MeanExpLevel - the mean of the normalized CDS expression levels for all samples in condition 1.
  • Condition2 MeanExpLevel - the mean of the normalized CDS expression levels for all samples in condition 2.

    • Note that "condition" is shown only in the case of two-group comparisons, while group name and time-point indicator are shown for single and multiple time-course designs.
  • Switching Times (single time course only) - a list of time points (by nubmer, as specified in the design matrix) where major isoform switching events occur.

The expression level for a CDS, in DIU, is the sum of the normalized levels for all transcripts with the same CDS.
  • Chr - chromosome where the gene is found.
  • Strand - chromosome strand.
  • Isoforms - number of isoforms for this gene.
  • Coding - protein coding status: YES, if there are any coding transcripts, NO otherwise.
  • Annotation Feature Columns - use the 'Add annotation feature column...' menu selection
