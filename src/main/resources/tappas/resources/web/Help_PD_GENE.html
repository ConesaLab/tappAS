Project Data: Genes

The Genes data table combines the transcript-level expression data with the functional annotation for the project. The following table columns are displayed by default:

  • Gene - gene identifier as provided in the annotation file
  • Gene Description - gene description, if any, as provided in the annotation file
  • Condition1 MeanExpLevel - the mean of the normalized transcript expression levels for all samples in condition 1
  • Condition2 MeanExpLevel - the mean of the normalized transcript expression levels for all samples in condition 2
The expression level for a gene is the sum of the normalized levels for all its transcripts
  • Chr - chromosome
  • Strand - chromosome strand
  • Isoforms - number of isoforms for this gene
  • Coding - protein coding status: YES, if there are any coding transcripts, or NO
  • Annotation Feature Columns - use the 'Add annotation feature column...' menu selection
