Project Data Visualization: GSEA Summary

he FEA Summary subtab contains multiple plots, along with supporting tables, summarizing the GSEA results. GSEA can be run using genes, CDS, or transcript data (hence the word Item is used refer to features, depending on the data type used). Note that enrichment can be tested for multiple functional categories simultaneously, all ow which will appear in both the tables and the summary plots. The display controls are described below by position, left to right.

Left panel: Items Summary data table - the table displays the total number of items included in the analysis. The number of ranked items is determined by the number of items contained in the specified ranked list. The following table columns are displayed for ranked and total items:

  • Item - item type
  • Count - number of items of given type
  • % - percent of total items of given type

Note that all ranked items in the list are included in this analysis so the values in both table rows will always be the same. They are both shown for consistency in using the total row.

Middle panel: Enriched Feature Count table - the table displays the total number of significantly enriched feature IDs in the test list per functional feature category. The following table columns are displayed:

  • Feature - feature category name.
  • Enriched Feature IDs - number of significantly enriched feature IDs for a given feature category.
  • Total Feature IDs - total number of feature IDs for given feature.

Right panel: Enriched Feature Percentages horizontal bar chart - the horizontal bar chart displays the percentage of enriched feature IDs (x axis) for each feature on the y axis. The total number of feature IDs tested for each functional feature category is used to calculate the percentages (i.e. data in the table in the middle panel).
